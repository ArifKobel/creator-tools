# creator-tools

## Overview:
creator-tools is a web application built using Golang, HTMX, Tailwind CSS, and FFmpeg to simplify the workflow of content creators. It offers a range of tools designed to handle tasks such as file convert, automatic subtitle generation, and more.

## Features:
- **File Conversion:** Convert files between different formats, such as video to audio, audio to video, and more.
- **Automatic Subtitle Generation:** Generate subtitles automatically for videos.
- **Customizable Options:** Fine-tune conversion and subtitle generation with customizable settings.

## Technologies:
- **Golang:** Backend development and integration.
- **HTMX:** For dynamic, AJAX-driven interactions on the frontend.
- **Tailwind CSS:** For efficient and responsive styling of the user interface.
- **FFmpeg:** For video processing and manipulation.
